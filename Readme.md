Cubot Note S 
===========
Device Tree N
------------------

Device Tree for Cubot Note S.

- MT6580 Chipset
- marshmallow 6.0 (3.18.19 Kernel)
- 5,5" 1280x720 display
- 4150MAh battery capacity
- 2GB RAM
